
# Desktop Pet

This is a script that can run in the background when you are working and a little pet sprite follows your cursor around.

Changes can be made to the sprite, or your own sprite can be added for your enjoyment. 



## Defualt Pets

* *Snail*, a slow creature that will follow you around, he is a chill dude.
* *Cat*, a naughty little guy, but he is too cute to be mad at. He will follow you around when you are busy, but when he gets bored he'll move around your desktop icons.
## Deployment

**To deploy this project run**

```bash
  python ./pet.py <Pet Name>
```
*Pet name isn't case sensitive.*

**To see list of avalible pets run**

```bash
    python ./pet.py list
```


## Documentation

**Add Sprite Pet** can be found at `AdditionalSprite.md` in the root directory.
